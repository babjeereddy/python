Input: n = 5

2. Count Even and Odd Numbers in a List

Input: [1, 2, 3, 4, 5, 6]
Output: Even: 3, Odd: 3

Input: num = 3

3 x 1 = 3
3 x 2 = 6
...
3 x 10 = 30
